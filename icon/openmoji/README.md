# OpenMoji Assets

This directory stores all imported OpenMoji SVG assets for MEMORYBLOX.

## Layout

- `svg/` — canonical folder for all OpenMoji SVG files.

## Import Rule

When importing additional OpenMoji icons:

1. Place SVG files directly in `icon/openmoji/svg/`.
2. Add attribution rows to `icon/ATTRIBUTION.csv`
   with `file_path` values under `icon/openmoji/svg/`.
3. Do not create batch-specific folders.
